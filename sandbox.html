Fixed padding top & bottom

This module has fixed top and bottom padding for the content per spec and does not change. This means that as the browser window opens horizontally, the image module gets pulled but the content module does not move so things start to clip.

Responsive padding top & bottom

This module has a responsive, percentage based padding for the content. As the browser grows horizontally the image module is pulled but also grows vertically since the content module is also growing. The downside is there can be a lot of whitespace on larger screens.
