This week's win

I've been meaning to switch this blogs code repo over from using Master to defaulting to Main for a while now. I did some quick searching and found a well written and simple blog post by Steven Mortimer. After reading his post, it took me about a minute to make the change and everything worked smoothly. The only gotcha was that when I deleted master, that temporarily disabled Github Pages which is built from master. It didn't automatically pick up the cutover and my site went down about five minutes after the switch. I had a hunch it was still looking for master so I checked the Pages GitHub Pages control panel and sure enough it had actually disabled it. I simply selected Main, clicked save, and my site came back online within about 10 seconds.
